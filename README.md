# 🎨 调色盘 - 专业配色方案工具

一个现代化的颜色搭配工具，输入任意颜色，自动生成 6 种专业配色方案，帮助设计师和开发者快速找到完美的颜色组合。

![主界面](main.png)

## ✨ 功能特点

### 🎯 核心功能

- **实时颜色输入**：支持 RGB 滑块调节、HEX 输入、颜色选择器三种方式
- **随机颜色生成**：一键生成随机颜色，激发灵感
- **6 种专业配色方案**：
  - **互补色**：色环上相对的两个颜色，产生强烈对比效果
  - **类比色**：色环上相邻的颜色，创造和谐统一的视觉效果
  - **三角色**：色环上等距的三个颜色，色彩丰富且平衡
  - **四角色**：色环上等距的四个颜色，提供丰富的色彩变化
  - **分裂互补色**：主色 + 互补色两侧的颜色，既有对比又保持和谐
  - **单色系**：同一色相的不同明度变化，简洁优雅

### 💾 数据管理

- **一键保存**：保存喜欢的配色方案到本地存储
- **配色历史**：查看已保存的所有配色方案
- **一键复制**：点击颜色卡片即可复制 HEX 色值

### 🎨 用户体验

- **响应式设计**：完美适配桌面和移动设备
- **暗色主题**：护眼的深色界面设计
- **实时预览**：所有配色方案同时展示，无需切换

## 🛠️ 技术栈

- **React 18** - 现代化的 UI 框架
- **TypeScript** - 类型安全的 JavaScript
- **Vite** - 极速的开发构建工具
- **Tailwind CSS** - 实用优先的 CSS 框架
- **TinyColor2** - 强大的颜色处理库

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173 查看应用

### 构建生产版本

```bash
npm run build
```

## 📖 配色方案说明

| 方案 | 描述 | 适用场景 |
|------|------|----------|
| 互补色 | 色环相对两色 | 需要强烈对比的设计 |
| 类比色 | 色环相邻色 | 自然、和谐的设计 |
| 三角色 | 等距三色 | 活泼、丰富的风格 |
| 四角色 | 等距四色 | 复杂、多变的配色 |
| 分裂互补色 | 主色+互补两侧 | 平衡对比与和谐 |
| 单色系 | 同色不同明度 | 简洁、专业的设计 |
